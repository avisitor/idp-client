Usage Example
<!-- Include the script --> <script src="/path/to/idp-auth.js"></script> <!-- Initialize authentication --> <script> IDPAuth.init({ idpUrl: 'https://your-idp.com', appId: 'your-app-id', tokenStorageKey: 'jwt_token', callbackUrl: window.location.href, bufferMinutes: 5 }); // Access token if needed const token = IDPAuth.getToken(); // Check if authenticated if (IDPAuth.isAuthenticated()) { console.log('User is authenticated'); } </script>
API Methods
IDPAuth.init(options) - Initialize authentication
IDPAuth.getToken() - Get valid JWT token (null if expired)
IDPAuth.isAuthenticated() - Check if user is authenticated
IDPAuth.logout(redirectUrl) - Clear token and logout
IDPAuth.processCallback() - Process IDP callback (for callback page)
IDPAuth.getTokenExpiration() - Get seconds until token expiration
